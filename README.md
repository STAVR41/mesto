# Проект: Место

### Проектная работа №4

Проект Mesto, интерактивная страница, куда можно добавлять фотографии, удалять их и ставить лайки.

Проекте используются такие технологии:
1. HTML
2. CSS
3. JS
4. BEM

Ссылка на проектную работу: https://stavr41.github.io/mesto/
